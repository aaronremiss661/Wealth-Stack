# PMS & AIF Fee Structures

## PMS Fee Models

### Model 1: Fixed Fee Only
| Component | Range |
|-----------|-------|
| Management fee | 1.5% - 3.0% of AUM per annum |
| Charged | Quarterly (0.375-0.75% per quarter) |
| Best for | Steady strategies, predictable costs |
| Downside | Manager incentive not aligned with performance |

### Model 2: Fixed + Performance Fee
| Component | Range |
|-----------|-------|
| Management fee | 0.5% - 1.5% of AUM per annum |
| Performance fee | 10% - 20% of profits above hurdle |
| Hurdle rate | 8% - 12% per annum |
| High-water mark | SEBI mandated (must use) |
| Crystallization | Annual or on exit |
| Best for | Growth strategies, aligns incentives |

### Model 3: Performance Fee Only
| Component | Range |
|-----------|-------|
| Management fee | 0% (or minimal) |
| Performance fee | 15% - 25% of all profits |
| High-water mark | Mandatory |
| Best for | Investors who want full alignment |
| Risk | Manager may take excessive risk for higher fee |

## High-Water Mark Explanation

- SEBI mandates high-water mark for performance fee calculation
- Performance fee charged only on returns ABOVE the previous highest NAV
- If NAV drops from 120 to 100, then rises to 115, no performance fee (still below 120)
- Performance fee only kicks in once NAV exceeds 120
- Prevents double-charging on recovery of previous losses

### Example:
| Period | NAV | Change | Performance Fee (20% above HWM) |
|--------|-----|--------|--------------------------------|
| Start | 100 | - | - |
| Year 1 end | 125 | +25% | 20% of 25 = Rs 5 per unit |
| New HWM | 120 (post-fee) | | |
| Year 2 end | 110 | -8% | Zero (below HWM) |
| Year 3 end | 135 | +23% | 20% of 15 (135-120 HWM) = Rs 3 per unit |
| New HWM | 132 | | |

## Additional PMS Costs

| Cost | Range | Notes |
|------|-------|-------|
| Brokerage | 0.03-0.05% per trade | Usually passed through |
| Demat charges | Rs 300-600/year | Custodian fees |
| GST on fees | 18% on management + performance fee | Significant addition |
| STT | 0.1% on equity delivery | On all trades |
| Exit load | 0-3% | If redeemed within 1-2 years |
| Capital gains tax | 12.5% LTCG / 20% STCG | On realized gains |

### True Cost Example (Rs 1 Cr PMS, 15% gross return)

| Component | Amount |
|-----------|--------|
| Gross return | Rs 15,00,000 |
| Management fee (1.5%) | -Rs 1,50,000 |
| Performance fee (15% of profit above 10% hurdle) | -Rs 75,000 |
| GST on fees (18%) | -Rs 40,500 |
| Brokerage + demat | -Rs 15,000 |
| Capital gains tax (12.5% LTCG on Rs 10L gains) | -Rs 1,25,000 |
| **Net to investor** | **Rs 10,94,500** |
| **Effective net return** | **~10.9%** |

## AIF Fee Structures

### Cat I (Venture Capital, Infra)
| Component | Typical |
|-----------|---------|
| Management fee | 2.0% - 2.5% |
| Carry (performance) | 20% above 8-10% hurdle |
| Catch-up | Yes (manager receives 100% after hurdle until carry achieved) |
| Clawback | Yes (if fund underperforms on aggregate) |
| Setup cost | 1-2% (one-time) |

### Cat II (PE, Debt)
| Component | Typical |
|-----------|---------|
| Management fee | 1.5% - 2.0% |
| Carry | 15-20% above hurdle |
| Hurdle | 8-12% |
| Fund life | 5-8 years (closed-ended) |

### Cat III (Hedge Funds)
| Component | Typical |
|-----------|---------|
| Management fee | 1.5% - 2.5% |
| Performance fee | 15-25% of profits |
| Hurdle | May or may not have |
| Lock-in | 1-3 years |
| Redemption | Monthly/quarterly |
