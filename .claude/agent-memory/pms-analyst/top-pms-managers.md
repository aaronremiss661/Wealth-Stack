## PMS Performance Benchmarking

### Where to Find Data
- SEBI PMS disclosure website: Monthly performance data
- PMS Bazaar: Third-party aggregator
- PMSAif.in: Comparison platform
- Direct from PMS manager: Audited returns

### Performance Comparison Rules
1. Always compare NET-of-fee returns (post all charges)
2. Use appropriate benchmark (Nifty 50 for large cap, Nifty 500 for multi cap)
3. Compare rolling returns (not point-to-point)
4. Minimum 3-year track record before evaluating
5. Check performance during drawdowns (2020 COVID, 2022 correction)
6. Account for survivorship bias in PMS data
