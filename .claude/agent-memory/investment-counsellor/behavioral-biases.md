# Behavioral Biases in Investing — Identification & Mitigation

## Cognitive Biases

### 1. Recency Bias
| Aspect | Details |
|--------|---------|
| Definition | Overweighting recent events while ignoring long-term patterns |
| Example | "Small caps fell 20% last quarter, I should exit all small caps" |
| Indian context | After 2020 crash: panic selling. After 2021 rally: overconfidence in small caps |
| Mitigation | Show rolling 5Y/10Y returns; automate SIPs; historical drawdown recovery data |

### 2. Loss Aversion
| Aspect | Details |
|--------|---------|
| Definition | Losses feel 2x more painful than equivalent gains feel good |
| Example | Holding loss-making stocks hoping to "break even" while selling winners early |
| Indian context | Refusing to book loss on PSU stocks or NFO investments that went wrong |
| Mitigation | Frame rebalancing as portfolio improvement, not "booking a loss"; tax-loss harvesting narrative |

### 3. Anchoring
| Aspect | Details |
|--------|---------|
| Definition | Over-relying on the first piece of information received |
| Example | "I'll buy Nifty when it comes back to 18,000" (anchored to past low) |
| Indian context | Anchoring to purchase price of property or stock |
| Mitigation | Focus on forward valuations, not historical prices; reframe with intrinsic value analysis |

### 4. Confirmation Bias
| Aspect | Details |
|--------|---------|
| Definition | Seeking information that confirms existing beliefs |
| Example | Only reading bullish analyst reports on a stock already owned |
| Mitigation | Present bear case alongside bull case; devil's advocate framework |

### 5. Overconfidence
| Aspect | Details |
|--------|---------|
| Definition | Overestimating one's ability to pick stocks or time markets |
| Example | Frequent F&O trading based on "gut feeling"; concentrated positions |
| Indian context | Post-2020 new retail traders (Zerodha wave) overtrading in options |
| Mitigation | Track actual returns vs benchmark; show data on active vs passive performance |

### 6. Herd Mentality
| Aspect | Details |
|--------|---------|
| Definition | Following the crowd without independent analysis |
| Example | IPO frenzy (applying because "everyone is applying"); NFO rush |
| Indian context | Paytm IPO (2021), SME IPO mania (2024) |
| Mitigation | Establish investment policy statement (IPS) before acting; checklist-driven process |

### 7. Disposition Effect
| Aspect | Details |
|--------|---------|
| Definition | Selling winners too early and holding losers too long |
| Example | Selling a 50% gainer while holding a 40% loser |
| Mitigation | Rules-based exit strategy; trailing stop losses; periodic position review |

### 8. Mental Accounting
| Aspect | Details |
|--------|---------|
| Definition | Treating money differently based on source or designation |
| Example | "Bonus money" invested aggressively; "salary savings" in FD only |
| Indian context | Treating PF/PPF as separate from MF portfolio in risk assessment |
| Mitigation | Unified portfolio view; consolidated net worth statement |

## Emotional Biases

### 9. Status Quo Bias
| Aspect | Details |
|--------|---------|
| Definition | Preference for current state; resistance to change |
| Example | Staying in high-expense regular MF plans because "it's working fine" |
| Mitigation | Show cost of inaction (regular vs direct delta over 20 years) |

### 10. Endowment Effect
| Aspect | Details |
|--------|---------|
| Definition | Overvaluing what you already own |
| Example | Valuing inherited property at 2x market value because of emotional attachment |
| Mitigation | "Would you buy this asset today at this price?" test |

## Bias Detection Framework

| Red Flag | Likely Bias | Counsellor Action |
|----------|------------|-------------------|
| "I want to sell everything" (after 10% fall) | Recency + loss aversion | Show historical recovery data |
| "I only want to invest in X sector" | Recency + confirmation | Present diversification benefits |
| "My friend made 50% in this stock" | Herd mentality | Return to IPS and risk profile |
| "I'll wait for market to fall more" | Anchoring | Discuss SIP/STP for systematic entry |
| "I can't sell this at a loss" | Loss aversion + disposition | Tax-loss harvesting benefit |
| "Markets always go up, I want 100% equity" | Overconfidence | Show 2008/2020 drawdowns in rupee terms |
| "FD is safest, I'll never lose money" | Status quo + mental accounting | Show negative real returns after tax and inflation |

## Nudge Techniques for Better Behavior

1. **Default SIP with step-up**: Auto-increase SIP by 10% annually
2. **Automate rebalancing**: Calendar-based, removes emotion
3. **Goal-linked investing**: Connect money to specific goals (not abstract "wealth creation")
4. **Cooling-off period**: 48-hour rule before making any major portfolio change
5. **Pre-commitment**: "If market falls 20%, I will invest Rs X additional" (written before the event)
