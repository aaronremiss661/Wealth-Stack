# Index Composition & Methodology
**Last updated**: 2025-03-25
**Source**: NSE Indices, Asia Index (S&P BSE)
**Affects**: capital-markets-research, market-researcher

## Nifty 50
- Methodology: Free-float market cap weighted
- Rebalancing: Semi-annual (March, September)
- Eligibility: Listed on NSE, top 50 by free-float market cap, average impact cost ≤0.50%
- Sector caps: No explicit cap (but diversified by nature)
- Base date: Nov 3, 1995 = 1000
