# Trust Types in India

## Legal Framework

Governed by: Indian Trusts Act, 1882 (private trusts) and relevant state Public Trusts Acts

## Private Trust

### Revocable Trust
| Feature | Details |
|---------|---------|
| Settlor control | Full — can modify, revoke, change beneficiaries |
| Tax treatment | Income taxed in settlor's hands (Section 61, 63) |
| Asset protection | Limited (creditors can reach assets) |
| Succession benefit | Avoids probate on death |
| Use case | During lifetime management, not for tax planning |

### Irrevocable Trust (Specific Beneficiaries)
| Feature | Details |
|---------|---------|
| Settlor control | None after creation (cannot revoke) |
| Tax treatment | Income taxed in beneficiary's hands at their slab rate |
| Condition | Beneficiaries and their shares must be specified in trust deed |
| Asset protection | Strong (assets legally separated from settlor) |
| Use case | Inter-generational transfer, protecting family assets |
| Stamp duty | Varies by state (typically 2-5% of trust property value) |

### Irrevocable Trust (Discretionary)
| Feature | Details |
|---------|---------|
| Trustee discretion | Trustee decides who gets what, when |
| Tax treatment | Maximum marginal rate (42.7%) on trust income |
| Flexibility | High — trustee can distribute based on need |
| Use case | Protecting spendthrift beneficiaries, unequal distribution |
| Downside | Highest tax rate, trustee liability |

## Public Trust (Charitable)

### Section 12A/12AB Registration
| Feature | Details |
|---------|---------|
| Purpose | Religious, educational, medical, or general public benefit |
| Tax status | Exempt from income tax if 85% of income applied to objects |
| Accumulation | Can accumulate up to 15% of income (invest per Sec 11(5)) |
| Registration | With Income Tax Commissioner (provisional 3 years, then regular 5 years) |
| Compliance | Annual IT return, audit report (Form 10B), accounts maintenance |
| Donor benefit | Donors get 80G deduction (50% or 100% depending on trust type) |

### FCRA Registration (for foreign contributions)
- Required if charitable trust receives donations from abroad
- Ministry of Home Affairs registration
- Separate FCRA bank account mandatory
- Annual FCRA return filing

## Trust Deed Key Clauses

| Clause | Purpose | Recommendation |
|--------|---------|----------------|
| Beneficiaries | Define who receives income/corpus | Name specific persons + class of persons |
| Trustee powers | Investment, distribution, management | Detailed but not unlimited |
| Successor trustee | Who takes over if trustee dies/resigns | Always specify 2-3 successors |
| Distribution rules | When, how much, to whom | Specify % or leave to trustee discretion |
| Protector clause | Independent overseer of trustee | Recommended for large trusts |
| Amendment clause | How trust deed can be modified | Limited amendment power for irrevocable trusts |
| Dissolution clause | What happens on termination | Assets go to specified beneficiaries |
| Investment clause | Where trust funds can be invested | Specify permissible asset classes |
| Conflict resolution | Dispute mechanism | Arbitration preferred over litigation |

## Trust vs HUF vs Company

| Feature | Private Trust | HUF | Private Company |
|---------|--------------|-----|-----------------|
| Tax rate | Slab (specific) or MMR (discretionary) | Slab rate | 22-25% |
| Asset protection | High (irrevocable) | Low (partition risk) | Moderate |
| Succession | Smooth (trust deed) | Automatic (Hindu law) | Complex (share transfer) |
| Privacy | High | Moderate | Low (RoC filings public) |
| Compliance | Moderate | Low | High |
| Flexibility | High | Limited | Moderate |
| Applicable to | All religions | Hindu/Jain/Sikh/Buddhist | All |
| Foreign assets | Possible (FEMA compliance) | Possible | Possible (with ODI rules) |

## Trust Taxation — Key Rules

| Section | Rule |
|---------|------|
| Section 56(2)(x) | Transfer to trust without consideration: taxable as gift if > Rs 50,000 |
| Section 61 | Revocable trust income = settlor's income |
| Section 63 | Transfer to spouse/minor through trust = clubbing applies |
| Section 160 | Trustee is representative assessee |
| Section 164 | Discretionary trust taxed at maximum marginal rate |
| Section 164A | Oral trust — same as discretionary |
| Section 161 | Specific trust — income taxed in beneficiary's hands |
