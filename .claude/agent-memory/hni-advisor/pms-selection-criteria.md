# PMS Selection Criteria & Due Diligence

## PMS Manager Evaluation Framework

### Quantitative Criteria (60% Weight)

| Criterion | Weight | Metric | Benchmark |
|-----------|--------|--------|-----------|
| Performance (3Y CAGR) | 15% | Net-of-fee return | > Nifty 500 + 3% alpha |
| Performance (5Y CAGR) | 15% | Net-of-fee return | > Nifty 500 + 2% alpha |
| Risk-adjusted return | 10% | Sharpe ratio (3Y) | > 1.0 |
| Drawdown management | 10% | Max drawdown vs benchmark | < benchmark drawdown |
| Consistency | 10% | Rolling 1Y quartile ranking | Top 2 quartiles >60% of time |

### Qualitative Criteria (40% Weight)

| Criterion | Weight | What to Check |
|-----------|--------|---------------|
| Fund manager pedigree | 10% | Experience, track record, tenure |
| Investment philosophy | 10% | Clear, repeatable, well-articulated process |
| AUM sustainability | 5% | Not too small (<100Cr) or too large for strategy |
| Operational quality | 5% | Custodian, auditor, reporting frequency |
| Fee structure fairness | 5% | Alignment with investor interest |
| Transparency | 5% | Portfolio disclosure, communication frequency |

## Fee Structure Comparison

| Fee Model | Typical Range | Suitable For |
|-----------|-------------|-------------|
| Fixed only | 1.5-2.5% of AUM | Steady, lower-risk strategies |
| Fixed + performance | 1-1.5% fixed + 10-15% above hurdle | Growth strategies |
| Performance only | 15-25% of profits (with high water mark) | High-conviction strategies |
| Hurdle rate | 8-10% (performance fee only on returns above this) | Aligns with investor minimum expectation |

### Fee Impact on Returns

| Fee Structure | Gross Return 15% | Net Return |
|--------------|------------------|------------|
| 2% fixed only | 15% | 13% |
| 1% fixed + 15% of profit above 10% | 15% | 13.25% |
| 20% of all profit | 15% | 12% |
| 1.5% fixed + 20% above 8% hurdle | 15% | 12.1% |

## PMS Due Diligence Checklist

### Pre-Investment
- [ ] SEBI PMS registration verification (SEBI website)
- [ ] Review SEBI PMS monthly disclosure data (performance, AUM)
- [ ] Fund manager interview (investment philosophy, process)
- [ ] Reference check with existing investors
- [ ] Verify custodian (should be reputed bank)
- [ ] Read investment management agreement (IMA) in detail
- [ ] Understand exit clauses and lock-in (if any)
- [ ] Check for any SEBI disciplinary actions against the manager
- [ ] Confirm high-water mark for performance fee
- [ ] Verify taxation handling (do they provide capital gains statement?)

### Ongoing Monitoring
- [ ] Monthly portfolio review (stock-level positions)
- [ ] Quarterly performance vs benchmark analysis
- [ ] Semi-annual fund manager interaction
- [ ] Annual fee audit (actual fees charged vs agreed structure)
- [ ] Check portfolio turnover (excessive churn = higher taxes)
- [ ] Monitor AUM changes (large outflows = red flag)
- [ ] Compare with peer PMS managers in same strategy
