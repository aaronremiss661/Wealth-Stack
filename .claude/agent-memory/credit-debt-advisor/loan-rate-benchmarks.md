# Loan Rate Benchmarks — FY 2025-26

## Current Interest Rate Environment

RBI Repo Rate: 6.25% (February 2025, cut from 6.50%)
Expected trajectory: Further cuts likely in 2025 (easing cycle)

## Home Loan

| Parameter | Details |
|-----------|---------|
| Rate range | 8.25% - 9.50% (floating) |
| Best available (top banks) | 8.25-8.50% (SBI, HDFC, ICICI for high credit score) |
| LTV (loan-to-value) | Up to 80% (loans up to Rs 30L: 90%) |
| Max tenure | 30 years |
| Processing fee | 0.25-0.50% of loan amount |
| Prepayment penalty | NIL for floating rate (RBI mandate) |
| Tax benefit (interest) | Section 24(b): Rs 2,00,000/year (self-occupied, old regime) |
| Tax benefit (principal) | Section 80C: Rs 1,50,000/year (old regime) |

## Loan Against Property (LAP)

| Parameter | Details |
|-----------|---------|
| Rate range | 9.00% - 12.00% |
| Best rates | 9.00-10.00% (banks, high-value property) |
| LTV | 50-70% of property market value |
| Max tenure | 15-20 years |
| Processing fee | 0.50-1.00% |
| Prepayment penalty | NIL for floating (banks), 2-4% for fixed |
| Use case | Business expansion, debt consolidation, large expenses |

## Loan Against Securities (LAS)

| Parameter | Details |
|-----------|---------|
| Rate range | 9.00% - 12.00% (bank), 10.00% - 14.00% (NBFC) |
| LTV — Equity shares | 50% of portfolio value |
| LTV — Mutual funds (equity) | 50% of NAV |
| LTV — Mutual funds (debt) | 65-80% of NAV |
| LTV — Fixed deposits | 80-90% |
| LTV — Insurance policies | 80-90% of surrender value |
| LTV — Gold | 75% of gold value |
| Tenure | Typically overdraft (renewable annually) |
| Processing fee | 0.50-1.00% |
| Margin call | If LTV exceeds limit, must pledge more or repay |

## Education Loan

| Parameter | Details |
|-----------|---------|
| Rate range | 8.50% - 13.00% |
| Up to Rs 7.5L | Usually no collateral (bank dependent) |
| Rs 7.5L - Rs 20L | Collateral may be required |
| Above Rs 20L | Collateral mandatory |
| Moratorium | Course period + 6-12 months |
| Tax benefit | Section 80E: Interest deduction (no limit, 8 years) |
| Repayment tenure | 5-15 years post-moratorium |

## Gold Loan

| Parameter | Details |
|-----------|---------|
| Rate range | 7.00% - 12.00% |
| LTV | Up to 75% of gold value (RBI mandate) |
| Tenure | 3-24 months (renewable) |
| Processing | Minimal (Rs 500-1,000) |
| Best use | Short-term cash need, emergency |
| Key players | Muthoot Finance, Manappuram, SBI, Bank of Baroda |

## Interest Rate Linked Benchmarks

| Benchmark | Current Rate | Linked To |
|-----------|-------------|-----------|
| RBI Repo Rate | 6.25% | All EBLR-linked loans |
| SBI EBLR | 8.25% | SBI home loans |
| MCLR (1-year, SBI) | 8.70% | Older floating loans |
| Base Rate (SBI) | 10.40% | Very old loans (pre-2016) |

**Key point**: EBLR (External Benchmark Lending Rate) loans adjust within 3 months of RBI rate change. MCLR-based loans adjust slower.
