# LAS (Loan Against Securities) & LAP (Loan Against Property) Rules

## Loan Against Securities (LAS)

### LTV (Loan-to-Value) Norms — RBI/SEBI Guidelines

| Collateral Type | Maximum LTV | Margin Required |
|----------------|-------------|----------------|
| Equity shares (listed, Group I) | 50% | 50% |
| Equity shares (listed, Group II/III) | 50% | 50% (may be higher for volatile stocks) |
| Equity shares (unlisted) | Not typically accepted | N/A |
| Equity mutual funds | 50% | 50% |
| Debt mutual funds | 65-80% | 20-35% |
| Government securities | 85-90% | 10-15% |
| Fixed deposits | 80-90% | 10-20% |
| Insurance policies (surrender value) | 80-90% | 10-20% |
| Gold (physical/ETF) | 75% | 25% |
| LIC endowment policies | 85-90% | 10-15% |

### LAS Interest Rates

| Lender Type | Rate Range | Processing Fee |
|------------|-----------|---------------|
| Banks (SBI, HDFC, ICICI) | 8.50-10.50% | 0.25-0.50% |
| NBFCs (Bajaj Finance, IIFL) | 9.50-12.00% | 0.50-1.00% |
| Brokerages (as NBFC) | 10.00-14.00% | 0.50-1.50% |
| Private banks (premium clients) | 8.00-9.50% | Negotiable |

### LAS Key Rules

| Rule | Requirement |
|------|-------------|
| Margin maintenance | Must maintain stipulated LTV at all times |
| Margin call trigger | When LTV exceeds limit due to security price drop |
| Margin call response time | Typically 2-3 business days to top up or repay |
| Pledge mechanism | Shares pledged in demat account (CDSL/NSDL) |
| Lien on MF | Lien marked with AMC/RTA |
| Income proof | Varies (some accept portfolio value as income proxy) |
| Tenure | Typically overdraft (1 year, renewable) or term loan (1-5 years) |
| Min loan amount | Rs 1-5 lakh (bank), Rs 5-10 lakh (NBFC) |
| Max loan amount | Up to Rs 20 Cr (bank), Rs 50 Cr+ (NBFC for HNI) |

### LAS Margin Call Process

| Step | Action |
|------|--------|
| 1 | Portfolio value drops below maintenance margin |
| 2 | Lender sends margin call notice (SMS/email) |
| 3 | Borrower must: (a) pledge additional securities, OR (b) partially repay loan |
| 4 | If not resolved in 2-3 days: Lender can sell pledged securities |
| 5 | Forced sale: Lender sells enough to restore LTV to initial level |

**Warning**: In sharp market corrections (e.g., COVID March 2020), margin calls can cascade. Borrowers should maintain a buffer of 10-15% above minimum LTV.

### LAS for Wealth Management Strategy

| Strategy | Use Case | Risk |
|----------|----------|------|
| Leverage to invest | Borrow against portfolio, invest in different asset | Market risk + interest cost |
| Tax-efficient liquidity | Avoid selling (and paying LTCG) by borrowing instead | Margin call risk |
| Bridge finance | Temporary cash need while waiting for proceeds | Low if short-term |
| Concentrated stock | Borrow against concentrated holding, diversify with proceeds | Double risk (stock + loan) |

## Loan Against Property (LAP)

### LTV Norms

| Property Type | Maximum LTV | Typical |
|--------------|-------------|---------|
| Residential (self-occupied) | 70-75% | 60-65% |
| Residential (rented) | 65-70% | 55-60% |
| Commercial (office/shop) | 60-65% | 50-55% |
| Industrial property | 50-60% | 40-50% |
| Plot / land | 50-60% | 40-50% |

### LAP Interest Rates

| Lender Type | Rate Range | Processing Fee |
|------------|-----------|---------------|
| Banks | 8.50-11.00% | 0.50-1.00% |
| HFCs | 9.00-12.00% | 0.50-1.00% |
| NBFCs | 10.00-14.00% | 1.00-2.00% |

### LAP Key Parameters

| Parameter | Details |
|-----------|---------|
| Max tenure | 15-20 years (bank), 10-15 years (NBFC) |
| Min loan | Rs 5-10 lakh |
| Max loan | Rs 5-10 Cr (bank), Rs 20 Cr+ (NBFC) |
| Prepayment penalty | NIL for floating rate (RBI rule for individuals) |
| End use | Business expansion, debt consolidation, education, medical, others |
| NOT allowed for | Speculation, investment in stocks (technically prohibited) |
| Valuation | Independent valuer appointed by lender |
| Legal check | Title search (30+ years), encumbrance certificate |
| Insurance | Property insurance mandatory (lender insured) |
| CERSAI registration | Mandatory (Central Registry of Securitisation) |

### LAP vs Home Loan vs Personal Loan

| Feature | Home Loan | LAP | Personal Loan |
|---------|----------|-----|---------------|
| Rate | 8.25-9.50% | 9.00-12.00% | 10.50-18.00% |
| Purpose | Purchase/construct house | Any (except speculation) | Any |
| Collateral | Property being purchased | Existing property | None |
| LTV | 75-90% | 50-70% | N/A |
| Tax benefit | Sec 24(b) + 80C | No (unless used for house) | No |
| Tenure | Up to 30 years | Up to 20 years | Up to 7 years |
| Prepayment | NIL (floating) | NIL (floating) | 2-5% penalty |
