# Wealth Report Templates & Presentation Best Practices

## Standard Wealth Report Structure

### 1. Portfolio Summary (Page 1)
| Element | Content |
|---------|---------|
| Total portfolio value | Current NAV with date |
| Absolute return (since inception) | Amount + percentage |
| Period returns | 1M, 3M, 6M, 1Y, 3Y, 5Y, SI |
| Benchmark comparison | Alpha/underperformance vs relevant index |
| Net inflows | Total invested vs current value |

### 2. Asset Allocation (Page 2)
| Visual | Content |
|--------|---------|
| Donut chart | Current allocation (equity/debt/gold/cash/alternatives) |
| Comparison bar | Current vs target allocation |
| Drift indicator | Traffic light (green/yellow/red) for each asset class |
| Rebalancing actions | Suggested trades to restore target |

### 3. Goal Tracker (Page 3)
| Element | Content |
|---------|---------|
| Goal name | Retirement, education, house, etc. |
| Target corpus | Inflation-adjusted future value |
| Current corpus | Allocated investments for this goal |
| Progress bar | % funded |
| On-track status | Green (>90% of plan), Yellow (70-90%), Red (<70%) |
| SIP adequacy | Current SIP vs required SIP |

### 4. Holdings Detail (Pages 4-5)
| Column | Details |
|--------|---------|
| Instrument name | Fund name / stock name |
| Category | Large cap / mid cap / debt / etc. |
| Invested amount | Total cost basis |
| Current value | Market value |
| Return (%) | Absolute and XIRR |
| Weight (%) | % of total portfolio |
| Holding period | Days/months held |
| Tax status | LTCG / STCG classification |

### 5. Transaction Summary (Page 6)
- Purchases, redemptions, SIP transactions for the period
- Dividend/interest received
- Tax harvesting transactions

## Chart Types for Financial Data

| Data Type | Best Chart | Why |
|-----------|-----------|-----|
| Asset allocation | Donut / pie chart | Shows proportions at a glance |
| Performance over time | Line chart | Shows trend and comparison |
| Return comparison (bar) | Horizontal bar chart | Easy comparison across categories |
| Goal progress | Progress bar / bullet chart | Intuitive progress visualization |
| Risk vs return | Scatter plot | Shows efficient frontier positioning |
| Sector exposure | Treemap | Shows relative size of sectors |
| SIP growth | Area chart | Shows accumulated value growth |
| Drawdown | Waterfall chart | Shows peak-to-trough movement |
| Income/expense flow | Sankey diagram | Shows money flow direction |

## Presentation Design Principles

### For HNI / UHNI Clients
- **Minimal, premium feel**: White space, restrained color palette
- **Dark background option**: Navy/black for executive presentations
- **Key numbers**: Large font, highlighted in accent color
- **Avoid**: Clutter, too many charts per slide, rainbow colors
- **Font**: Sans-serif (clean: Inter, Helvetica, Calibri)
- **One insight per slide**: Each slide answers one question

### Color Palette for Financial Reports

| Color Use | Recommended |
|-----------|------------|
| Positive returns / gains | Green (#22C55E) or Teal |
| Negative returns / losses | Red (#EF4444) or Coral |
| Primary data | Navy (#1E3A5F) or Dark Blue |
| Secondary data | Gray (#6B7280) |
| Accent / CTA | Gold (#D4AF37) or Deep Orange |
| Background | White (#FFFFFF) or Off-white (#F9FAFB) |

### Data Visualization Rules
1. **Start Y-axis at zero** (for bar charts) to avoid misleading
2. **Use consistent scales** across comparison charts
3. **Label key data points** directly on chart (not just in legend)
4. **Limit colors**: Max 5-6 distinct colors per chart
5. **Show benchmark**: Always overlay benchmark for context
6. **Time period**: Always label the time period clearly
7. **Source**: Cite data source and "as of" date

## Formatting Standards
- All amounts in INR with Indian numbering (lakhs, crores)
- Tables for data, charts described in markdown (agent generates structured data for charting)
- Color coding: Green (on track), Yellow (watch), Red (action needed)
- Date format: DD-MMM-YYYY
- Font: Keep clean, professional — avoid excessive decoration

## Slide-by-Slide Template for Annual Review Deck

| Slide # | Title | Content |
|---------|-------|---------|
| 1 | Cover | Client name, report period, advisor name |
| 2 | Executive summary | 3-4 key highlights in bullet points |
| 3 | Market overview | Key index returns, macro events (brief, 1 slide) |
| 4 | Portfolio summary | Total value, returns, benchmark comparison |
| 5 | Asset allocation | Current vs target (donut + bar) |
| 6 | Goal tracker | All goals with progress bars |
| 7 | Equity holdings | Top 10 stocks/funds, sector exposure |
| 8 | Fixed income | Debt holdings, yield, duration |
| 9 | Cost summary | Total fees, expense ratios, taxes paid |
| 10 | Tax summary | LTCG/STCG, harvesting done, tax saved |
| 11 | Action plan | Rebalancing, new SIPs, upcoming actions |
| 12 | Outlook & next steps | Not predictions — process, timeline, next review |
| 13 | Disclaimers | Regulatory disclaimers (mandatory) |
