## Sector Cyclicality

| Sector | Cycle Type | Key Drivers |
|--------|-----------|-------------|
| BFSI | Interest rate cycle | RBI repo rate, credit growth, NPA cycle |
| IT | Global tech spend cycle | US GDP, tech budgets, USD/INR |
| Pharma | Defensive / counter-cyclical | US FDA approvals, domestic demand |
| FMCG | Defensive | Rural demand, monsoon, inflation |
| Auto | Consumer discretionary cycle | Interest rates, fuel prices, EV transition |
| Metals | Commodity super-cycle | China demand, global commodity prices |
| Real Estate | Interest rate + liquidity cycle | Home loan rates, urbanization, RERA |
| Infra / Capital Goods | Government capex cycle | Union budget allocation, state spending |
| Telecom | Pricing cycle | ARPU trends, 5G rollout, consolidation |
| Energy / Power | Regulatory + demand cycle | Coal prices, renewable transition, demand growth |

## Sector Rotation Patterns (India)

### Early Recovery (Post-Recession)
Best performers: BFSI, Auto, Real Estate, Small Caps
Worst performers: FMCG, Pharma, Utilities

### Mid-Cycle Expansion
Best performers: IT, Capital Goods, Infrastructure, Mid Caps
Worst performers: Defensive sectors

### Late Cycle / Overheating
Best performers: Metals, Energy, Commodities
Worst performers: Rate-sensitive (BFSI, Real Estate, Auto)

### Recession / Slowdown
Best performers: FMCG, Pharma, Utilities (defensives)
Worst performers: Cyclicals (Metals, Auto, Real Estate)
