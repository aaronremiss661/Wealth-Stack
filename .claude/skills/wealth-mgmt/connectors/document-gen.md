# Document Generator Connector

## Purpose
Generate professional client-facing reports in HTML format from portfolio data and analysis results.

## Report Types

### 1. Portfolio Summary Report
Single-page overview of current holdings, allocation, and performance.

### 2. Quarterly Review Report
Multi-section report with performance, attribution, outlook, and recommendations.

### 3. Tax Planning Report
Year-end tax optimization summary with harvesting opportunities.

### 4. Financial Plan Report
Goal tracking, retirement projections, and planning milestones.

## Template System

Templates are stored in `${CLAUDE_SKILL_DIR}/templates/` and use simple `{{variable}}` substitution.

## Generation Process

1. Load data from `~/.wealth-mgmt/` (portfolios, reports, risk profile)
2. Build HTML sections from analysis data
3. Substitute variables in template
4. Write to `~/.wealth-mgmt/reports/<report-type>-<date>.html`
5. Optionally open in browser: `open <report-path>`

## Currency Formatting

For INR: `₹X,XX,XXX` (Indian numbering system — lakhs and crores)
For USD: `$X,XXX,XXX`

```bash
# Format INR with Indian commas
python3 -c "
def format_inr(amount):
    s = str(int(amount))
    if len(s) <= 3: return '₹' + s
    last3 = s[-3:]
    rest = s[:-3]
    groups = []
    while rest:
        groups.insert(0, rest[-2:] if len(rest) >= 2 else rest)
        rest = rest[:-2]
    return '₹' + ','.join(groups) + ',' + last3
print(format_inr(1250000))  # ₹12,50,000
"
```
