# WealthStack — AI Wealth Management for India

## Skills (Slash Commands)

### Core
| Command | Description |
|---------|-------------|
| `/wealthstack:portfolio` | Portfolio analysis — holdings, allocation, performance |
| `/wealthstack:risk-assess` | Risk profiling, stress testing, VaR, Sharpe ratio |
| `/wealthstack:tax-optimize` | LTCG/STCG harvesting, lot optimization, old vs new regime |
| `/wealthstack:rebalance` | Drift analysis, target allocation, trade generation |
| `/wealthstack:client-report` | Professional HTML portfolio reports |
| `/wealthstack:market-brief` | Nifty/Sensex, RBI policy, FII/DII flows, sector analysis |
| `/wealthstack:compliance` | SEBI suitability, concentration limits, documentation |
| `/wealthstack:financial-plan` | Retirement, estate, education, Monte Carlo projections |
