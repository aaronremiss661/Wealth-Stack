Get Started

Three steps to your
AI wealth desk

Clone

Get the repository on your machine.

git clone https://github.com/revanupoju/Wealth-Stack.git

Load

Start Claude Code with the plugin.

claude --plugin-dir ./Wealth-Stack

Ask

Start with any skill or just describe what you need.

/wealthstack:goal-planner "retire at 50"
Self-Improving

Gets smarter with
every conversation

Inspired by Karpathy's autoresearch. Every agent has persistent memory. New tax rules, regulation changes, edge cases — they're saved and used in future responses.

Run /wealthstack:research for active knowledge updates. Run /wealthstack:audit to clean stale facts.
